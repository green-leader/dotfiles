# Using this repo
It's been configured to run with [chezmoi](https://www.chezmoi.io/)

## install vim plugins
```bash
# clone vundle
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
# install plugins
vim +PluginInstall +qall
# show installed plugins
vim +PluginList
```
